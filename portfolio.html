  • 이미지 슬롯 시스템: 복잡한 다중 이미지 관리 로직 구현 (수정 시 순서 유지)

🛠️ 풀스택 개발 능력

백엔드 개발

  • RESTful API 설계: FastAPI로 총 25개 이상의 엔드포인트 구현
  • 데이터베이스 설계: SQLAlchemy ORM을 활용한 관계형 DB 모델링
  • 인증/보안: JWT 토큰 기반 인증, Argon2 패스워드 해싱
  • 파일 업로드: Cloudinary API 연동 및 이미지 최적화

서버 배포 & 운영

  • CI/CD 구축: GitHub → Vercel/Render 자동 배포 파이프라인
  • 환경 변수 관리: 로컬/프로덕션 환경 분리 및 보안 설정
  • 모니터링: UptimeRobot으로 서버 헬스체크 및 가동률 관리
  • CORS 정책: 프로덕션 환경에 맞는 보안 설정
  • CDN 활용: Cloudinary로 정적 리소스 최적화

💡 문제 해결 사례

  • Render 슬립 문제: Health Check 엔드포인트 + UptimeRobot으로 24/7 가동 구현
  • 이미지 영속성: 로컬 스토리지의 ephemeral 특성 파악 → Cloudinary 마이그레이션
  • CORS 에러: allow_origin_regex로 동적 도메인 대응
  • 405 Method Not Allowed: HEAD 메서드 지원으로 모니터링 최적화
